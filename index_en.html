Raop.js

RAOP is a simple JavaScript library that provides an object-based AOP.

Download .zip Download .tar.gz View on GitHub

Installation

  • nodejs

    $ npm install raop

  • browser

    <script type="text/javascript" src="/path/to/raop.min.js"></script>

Usage

Basic

raop.weave

raop.weave({aopTarget}, {pointcut}, {adviceType}, {advice})

  • aopTarget

    object to apply aop

  • pointcut

    It is a method to apply aop in aopTarget.
    RegExp or function can be. If you use function pointscut, the arguments are as follows.

    Method Name
    each aopTarget's public method name

  • adviceType

    advice timing. raop.Aspect.(BEFORE|AFTER|EXCEPTION|AROUND)

    raop.Aspect.BEFORE
    Before running the actual method.
    raop.Aspect.AFTER
    AFTER running the actual method.
    raop.Aspect.EXCEPTION
    When any object is thrown. You can access object thrown to use "exception" property name.
    raop.Aspect.AROUND
    Advise running and the actual method is passed as the first argument. "options" argument is the second argument.

  • advice

    function to advice.
    Arguments is a single object. The object properties are as follows

    args
    target
    todo
    advice
    type

Simple Logging with AOP Example

var calculator = {
    plus: function(a, b) {
        return a + b;
    },
    minus: function(a, b) {
        return a - b;
    }
};

var logAdvice = function(options) {
    console.log("method " + options.method + " execute");
};

// all method aop apply, type BEFORE
calculator = raop.weave(
    calculator,
    function() {
        return true;
    },
    raop.Aspect.AdviceType.BEFORE,
    logAdvice
);

// console.log("method plus execute")
var result = calculator.plus(1, 1);
