Raop.js

RAOP is a simple JavaScript library that provides an object-based AOP.

Download .zip Download .tar.gz View on GitHub

Installation

  • nodejs

    $ npm install raop

  • browser

    <script type="text/javascript" src="/path/to/raop.min.js"></script>

Usage

Basic

raop.weave

raop.weave({aopTarget}, {pointcut}, {adviceType}, {advice})

  • aopTarget

    object to apply aop

  • pointcut

    method to apply aop in aopTarget

  • adviceType

    advice timing. raop.Aspect.(BEFORE|AROUND|AFTER|EXCEPTION)

  • advice

    function to advice

Simple Logging with AOP Example

var calculator = {
    plus: function(a, b) {
        return a + b;
    },
    minus: function(a, b) {
        return a - b;
    }
};

var logAdvice = function(options) {
    console.log("method " + options.method + " execute");
};

// all method aop apply, type BEFORE
calculator = raop.weave(
    calculator,
    function() {
        return true;
    },
    raop.Aspect.AdviceType.BEFORE,
    logAdvice
);

// console.log("method plus execute")
var result = calculator.plus(1, 1);
